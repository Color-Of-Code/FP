# Lawvere Theories

A **Lawvere theory** is a categorical presentation of an algebraic theory (like monoid, group, ring,
or module): a small category $\mathbb{T}$ with finite products whose objects are natural numbers
$0, 1, 2, \ldots$ and where $n$ is the $n$-fold product of the object $1$. A **model** of the theory
is a product-preserving functor $\mathbb{T} \to \mathbf{Set}$. In programming, Lawvere theories
formalise what type classes do: they specify the operations and equations, and instances are the
models.

## Definition

### Lawvere theory

A **Lawvere theory** is a category $\mathbb{T}$ such that:

- Objects are natural numbers $\{0, 1, 2, \ldots\}$
- The object $n$ is the $n$-fold product $1^n$ (i.e. $1 \times 1 \times \cdots \times 1$)
- $\mathbb{T}$ has all finite products

A morphism $n \to 1$ in $\mathbb{T}$ is an **$n$-ary operation** of the theory. A morphism $n \to m$
is an $m$-tuple of $n$-ary operations (by the product structure).

### Models and semantics

A **model** (or **algebra**) of $\mathbb{T}$ is a functor $M : \mathbb{T} \to \mathbf{Set}$ that
**preserves finite products**:

$$M(n) \cong M(1)^n$$

The set $M(1)$ is the **carrier**; the morphisms $n \to 1$ become functions $M(1)^n \to M(1)$.

A **homomorphism** of models is a natural transformation between two models.

### The category of models

$$\mathbf{Mod}(\mathbb{T}) = [\mathbb{T}, \mathbf{Set}]_{\mathrm{fp}}$$

(product-preserving functors). This is equivalent to the category of algebras for the monad
associated with the theory.

### Examples

| Theory        | Operations                                         | Equations                |
| ------------- | -------------------------------------------------- | ------------------------ |
| Monoid        | `•: 2→1`, `e: 0→1`                                 | Associativity, unit laws |
| Group         | `•: 2→1`, `e: 0→1`, `inv: 1→1`                     | Monoid + inverse laws    |
| Ring          | `+: 2→1`, `0: 0→1`, `×: 2→1`, `1: 0→1`, `neg: 1→1` | Ring axioms              |
| Set (trivial) | Only projections                                   | None                     |

## Laws

A model $M$ must preserve products: $M(m \times n) \cong M(m) \times M(n)$. The equations of the
algebraic theory are encoded as **commutative diagrams** in $\mathbb{T}$ (pairs of parallel
morphisms that are required to be equal).

**Soundness and completeness**: the models of $\mathbb{T}$ are exactly the algebras satisfying the
equational theory. No models exist that violate the diagram equations.

## FP Analog

| CT concept                              | Haskell / FP                                                         |
| --------------------------------------- | -------------------------------------------------------------------- |
| Lawvere theory $\mathbb{T}$             | A type class (signature + laws)                                      |
| $n$-ary operation $n \to 1$             | A type-class method of arity $n$                                     |
| Model $M : \mathbb{T} \to \mathbf{Set}$ | A type-class instance                                                |
| Carrier set $M(1)$                      | The type that has the instance                                       |
| Homomorphism of models                  | A function preserving the operations (structure map)                 |
| Theory of monoids                       | `class Monoid a where mempty :: a; mappend :: a -> a -> a` + laws    |
| Theory of functors                      | `class Functor f where fmap :: (a -> b) -> f a -> f b` + laws        |
| `Free` monad for a theory               | The free model: `Free f a` over a signature functor `f`              |
| `Monad` from a Lawvere theory           | Every Lawvere theory generates a monad whose algebras are its models |
| `FinSet^op → Set`                       | The simplest Lawvere theory: the theory of sets (no operations)      |

The correspondence between Lawvere theories and monads on $\mathbf{Set}$ (for finitary monads) is
one of the deepest results in categorical algebra: **finitary monads = Lawvere theories**. The monad
for a theory $\mathbb{T}$ sends a set $X$ to the free model generated by $X$.
