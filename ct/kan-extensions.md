# Kan Extensions

**Kan extensions** are the most general construction in category theory. Saunders Mac Lane wrote:
"The notion of Kan extensions subsumes all the other fundamental concepts of category theory."
Products, limits, adjunctions, Yoneda — all are special cases of Kan extensions. In programming they
appear as indexed/graded data types, datatype-generic programming, and the efficient representation
of monads and comonads.

## Definition

Let $K : \mathcal{A} \to \mathcal{B}$ and $F : \mathcal{A} \to \mathcal{C}$.

### Right Kan extension

The **right Kan extension** $\mathrm{Ran}_K F : \mathcal{B} \to \mathcal{C}$ is the right adjoint to
precomposition with $K$:

$$[\mathcal{A}, \mathcal{C}](G \circ K,\; F) \;\cong\; [\mathcal{B}, \mathcal{C}](G,\; \mathrm{Ran}_K F)$$

Concretely (when $\mathcal{C} = \mathbf{Set}$):

$$(\mathrm{Ran}_K F)(b) \;=\; \int_a \mathbf{Set}(\mathcal{B}(b, K a),\; F a)$$

### Left Kan extension

The **left Kan extension** $\mathrm{Lan}_K F : \mathcal{B} \to \mathcal{C}$ is the left adjoint to
precomposition with $K$:

$$[\mathcal{A}, \mathcal{C}](F,\; G \circ K) \;\cong\; [\mathcal{B}, \mathcal{C}](\mathrm{Lan}_K F,\; G)$$

Concretely (when $\mathcal{C} = \mathbf{Set}$):

$$(\mathrm{Lan}_K F)(b) \;=\; \int^a \mathcal{B}(K a, b) \times F a$$

### Kan extensions recover other constructions

| Construction                               | As a Kan extension                                                                |
| ------------------------------------------ | --------------------------------------------------------------------------------- |
| Limit of $F : \mathcal{A} \to \mathcal{C}$ | $\mathrm{Ran}_{!} F$ where $! : \mathcal{A} \to \mathbf{1}$ is the unique functor |
| Colimit of $F$                             | $\mathrm{Lan}_{!} F$                                                              |
| Right adjoint of $K^*$ (precomposition)    | $\mathrm{Ran}_K \mathrm{Id}$                                                      |
| Yoneda lemma                               | $F \cong \mathrm{Ran}_{\mathrm{Id}} F$                                            |
| Density comonad                            | $\mathrm{Ran}_K K$                                                                |

## Laws

**Universality**: the counit / unit maps

$$\varepsilon : \mathrm{Ran}_K F \circ K \Rightarrow F \qquad \eta : F \Rightarrow \mathrm{Lan}_K F \circ K$$

are natural transformations witnessing the universal property. Any natural transformation
$G \circ K \Rightarrow F$ factors uniquely through $\varepsilon$.

## FP Analog

| CT concept                    | Haskell                                                             |
| ----------------------------- | ------------------------------------------------------------------- |
| $\mathrm{Ran}_K F$            | `newtype Ran k f b = Ran { runRan :: forall a. (b -> k a) -> f a }` |
| $\mathrm{Lan}_K F$            | `data Lan k f b = forall a. Lan (k a -> b) (f a)`                   |
| Right Kan along `Identity`    | Yoneda: `Ran Identity f a ≅ f a`                                    |
| Right Kan along `Const`       | Limit: `Ran (Const ()) f () ≅ forall a. f a`                        |
| Left Kan along `Identity`     | Co-Yoneda: `Lan Identity f a ≅ f a`                                 |
| `Density k a` = `Ran k k a`   | Density comonad: the comonad generated by any functor `k`           |
| `Codensity m a` = `Ran m m a` | Codensity monad: improves asymptotic performance of free monads     |
| `fromRan . toRan = id`        | Yoneda reduction at work in the Kan representation                  |

The **codensity monad** `Codensity m` has the same `Monad` interface as `m` but left-associates all
binds (like the continuation monad), removing the quadratic cost of naive left-associated sequences
in free monads. This is a direct application of right Kan extensions.
