# Ends and Coends

An **end** is a limit-like construction that internalises the notion of a "uniform family" indexed
by an object appearing both covariantly and contravariantly. A **coend** is the dual — a
colimit-like construction. Together they are the categorical basis of:

- `forall c. P c c` (ends in Haskell, via universal quantification)
- `exists c. P c c` (coends, via existential quantification)
- Profunctor composition and the `Category` instance for profunctors
- The natural-transformation formula $\mathrm{Nat}(F, G) = \int_c \mathrm{Set}(Fc, Gc)$

## Definition

### Dinatural transformations

Let $P : \mathcal{C}^{op} \times \mathcal{C} \to \mathbf{Set}$. A **dinatural transformation** from
$P$ to a constant functor $K$ is a family $\alpha_c : P(c, c) \to K$ such that for every
$f : c \to c'$:

$$\alpha_{c'} \circ P(\mathrm{id}, f) = \alpha_c \circ P(f, \mathrm{id})$$

(The two ways of moving along $f$ commute.)

### End

The **end** of $P$ is a universal dinatural transformation: an object
$\int_c P(c, c) \in \mathbf{Set}$ together with projections $\pi_c : \int_c P(c,c) \to P(c,c)$ such
that any dinatural transformation from $P$ factors uniquely through $\int_c P(c,c)$.

Concretely in $\mathbf{Set}$:

$$\int_c P(c,c) \;=\; \{\, (x_c)_{c \in \mathcal{C}} \mid x_c \in P(c,c),$$

$$P(f,\mathrm{id})(x_{c'}) = P(\mathrm{id},f)(x_c) \;\forall f : c \to c' \,\}$$

### Coend

The **coend** of $P$ is the universal cocone: $\int^c P(c,c)$ is the quotient of
$\bigsqcup_c P(c,c)$ by the equivalence generated by $P(\mathrm{id},f)(x) \sim P(f,\mathrm{id})(x)$.

### Key formulas

**Natural transformations as an end:**

$$\mathrm{Nat}(F, G) \;=\; \int_c \mathbf{Set}(F c,\; G c)$$

**Profunctor composition as a coend:**

$$(Q \circ P)(a, c) \;=\; \int^b P(a, b) \times Q(b, c)$$

**Yoneda lemma via coend:**

$$F a \;\cong\; \int^c \mathcal{C}(c, a) \times F c$$

## Laws

An end / coend, if it exists, is unique up to unique isomorphism (by universality, as with all
limits/colimits).

**Fubini for ends/coends**: ends and coends over multiple variables commute (if they all exist):

$$\int_c \int_d P(c, d, c, d) \;\cong\; \int_{c,d} P(c,d,c,d) \;\cong\; \int_d \int_c P(c,d,c,d)$$

## FP Analog

| CT concept                                            | Haskell                                                              |
| ----------------------------------------------------- | -------------------------------------------------------------------- |
| End $\int_c P(c,c)$                                   | `forall c. P c c` — rank-2 universally quantified type               |
| Coend $\int^c P(c,c)$                                 | `data Coend p = forall c. Coend (p c c)` — existential               |
| $\mathrm{Nat}(F, G) = \int_c \mathrm{Set}(Fc, Gc)$    | `forall a. f a -> g a` — polymorphic function                        |
| Profunctor composition via coend                      | `(q .  p) a c = exists b. (p a b, q b c)`                            |
| `type (~>) f g = forall a. f a -> g a`                | A natural transformation as an end                                   |
| `Day f g a = exists b c. f b -> g c -> (b -> c -> a)` | Day convolution as a coend                                           |
| Van Laarhoven lens                                    | `forall f. Functor f => (a -> f b) -> s -> f t` — end over `Functor` |
| Profunctor optic                                      | `forall p. Profunctor p => p a b -> p s t` — end over `Profunctor`   |

The Haskell `RankNTypes` extension provides `forall`, which corresponds exactly to ends. Existential
types correspond to coends. This is why profunctor optics and van Laarhoven lenses are naturally
expressed as `forall` types.
